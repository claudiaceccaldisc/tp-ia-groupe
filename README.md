# TimeTravel Agency

Webapp interactive développée dans le cadre du module IA M1 d'Ynov.

## Stack technique
- React + Vite
- Tailwind CSS
- Mistral AI (mistral-small)
- Déploiement Vercel

## Fonctionnalités
- Landing page immersive avec vidéo
- 3 destinations temporelles
- Quiz personnalisé
- Chatbot IA conversationnel

## IA utilisées
- Génération initiale du code : Bolt.new
- Chatbot : Mistral Small API

## Assets
Images et vidéo générées lors du module précédent.

## Déploiement
Vercel (gratuit)
